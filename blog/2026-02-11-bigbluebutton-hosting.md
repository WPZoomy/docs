## WordPress Plugin Configuration

To integrate BigBlueButton with WordPress:

1. Install the BigBlueButton plugin
2. Go to **Settings** → **BigBlueButton**
3. Add your server URL
4. Add your secret key
5. Configure room settings

:::tip Pro Tip
Always test the hosting service with a free trial before committing to a paid plan. Check bandwidth limits and concurrent user capacity.
:::

## Frequently Asked Questions

**Q: Is BigBlueButton really free?**  
A: Yes, the software is open-source and free. You only pay for hosting if you don't host it yourself.

**Q: How many users can join a session?**  
A: This depends on your hosting plan and server resources. Most providers support 50-200 concurrent users per room.

**Q: Can I record sessions?**  
A: Yes, BigBlueButton supports session recording, but check if your hosting plan includes storage for recordings.
