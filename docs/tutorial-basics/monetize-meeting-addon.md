# Monetize Meeting ADD On

## Prerequisites

1. [**Zoom WordPress Plugin**](https://wpzoomy.com/) (at least v4.17.0)
2. [**Easy Digital Downloads**](https://easydigitaldownloads.com/?ref=5027)

## Configuration Steps

1. Purchase the addon from below. You will receive a license key and plugin file via email.
2. Upload the plugin zip file to your WP site.
3. Add the License key to **Zoom Meetings -> Extensions** -> **Monetize Meeting Addon** and click activate. This will enable you to use the addon on your site.
4. Navigate to WP Dashboard -> Zoom Meetings, click the Create Download button for your preferred meeting.
5. Design the download page using your favorite page builder. That's it! Now share this page with your users. Once they purchase the download they will have access to the restricted meeting page that is created automatically with the download.
